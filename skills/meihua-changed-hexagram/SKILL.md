# 梅花易数算变卦

这个 Skill 单独负责第三步：

- 做计算，算变卦

## 规则写死

先读取：

- `../../docs/trigram-visual-reference.md`

变卦的算法固定是：

1. 先把两个数字相加
2. 再除以 `6`
3. 余数就是`动爻`
4. 如果余数为 `0`，按 `6` 论
5. 从下往上数第几爻动
6. 阴阳互变，得到变卦

## 绝对不要写错

不要把这一步写成：

- `7 ÷ 3`
- `7 除 3`
- 把两个数字直接相除

正确写法必须是类似：

- `7 + 3 = 10`
- `10 ÷ 6，余 4`
- `动爻：第四爻`
- `第四爻按自下而上计数`
- `如果本爻原本是阴，就要变阳；如果原本是阳，就要变阴`

## 图示硬规则

算变卦时，必须同时对齐三件事：

1. 动爻前的六爻图
2. 动爻变化后的六爻图
3. 变化后上卦 / 下卦对应的卦名

并且要牢牢记住：

- `阳` = 一整横
- `阴` = 中间断开

严禁出现这种错乱：

- 动爻明明是阴变阳，却写成阳变阴
- 图示已经变成某卦，名字却还是另一个卦
- 上下三爻图和卦名不一致

## 73 案例校准

如果案例是 `73`，本卦是 `山火贲`：

- 第四爻必须按“自下而上”数
- 第四爻原本是 `阴`
- 变化后必须变成 `阳`
- 因此上卦由 `艮` 变成 `离`
- 最后得到 `离为火`

如果图里把这一步画成“阳变阴”，直接判错。

### 八卦图示固定表

- 乾：阳 / 阳 / 阳
- 兑：阴 / 阳 / 阳
- 离：阳 / 阴 / 阳
- 震：阴 / 阴 / 阳
- 巽：阳 / 阳 / 阴
- 坎：阴 / 阳 / 阴
- 艮：阳 / 阴 / 阴
- 坤：阴 / 阴 / 阴

## 输入

- 第一位数字
- 第二位数字
- 本卦六爻结构

## 输出

- 两数之和
- 除以 6 后的余数
- 动爻位置
- 变卦结果
