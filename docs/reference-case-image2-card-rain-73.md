# Image2 教学卡参考案例

## 案例名称

`明天会不会下雨？ / 数字 73`

## 用途

这是当前插件中，用来约束 `image2 教学卡` 版式方向的参考案例。

它的作用不是决定事实，而是固定：

- 结构
- 版式感
- 教学感
- 信息组织方式

## 本案例事实

- 问题：明天会不会下雨？
- 数字：73
- 上卦：7 -> 艮 -> 土
- 下卦：3 -> 离 -> 火
- 本卦：山火贲
- 动爻：第四爻
- 变卦：离为火
- 体：艮（土）
- 用：离（火）
- 关系：用生体
- 结论：偏不下雨

## 参考结构

四步固定：

1. 明确问题，给数字
2. 做计算，算本卦
3. 做计算，算变卦
4. 做解析

并固定带一块：

- 八卦图形对照区

## 第二步重点

第二步必须有一种“一眼看懂”的学习效果：

- 数字如何对应八卦
- 八卦如何对应五行
- 八卦名称如何对应图形
- 为什么 7 是艮、是土
- 为什么 3 是离、是火

## 第四步重点

第四步必须有：

- 八卦五行表
- 完整五节点五行生克图
- 体用关系
- 最终结论

这里的五行生克图必须完整包含：

- 金
- 水
- 木
- 火
- 土

未来生成新的 image2 教学卡时：

- 可以沿用这个案例的版式方向
- 但不能沿用这个案例的具体卦象事实
- 新案例必须以自己的 `result.json` 为准
