# 梅花易数方法论参考

本文件用于把你提供的《梅花易数.png》沉淀为插件内的长期参考方法论。

## 这套方法的定位

- 用来辅助决策
- 用来处理“不确定”
- 重点不是玄谈，而是形成一套能重复使用的操作流程
- 结论服务于行动，而不是只做概念解释

## 使用边界

- 小事不占
- 跟自己没关系的事不占
- 问题尽量具体
- 起卦前先把问题想清楚

## 四步主流程

1. 明确问题，给数字
2. 做计算，算本卦
3. 做计算，算变卦
4. 做解析

## 第一步：明确问题，给数字

- 先在脑海中明确一件事
- 问题越具体，后面的判断越可用
- 默认取两位数字

例：

- 今年这个合作能不能顺利推进？
- 明天会不会下雨？
- 这件事值不值得做？

## 第二步：做计算，算本卦

### 上卦算法

- 第一位数字 ÷ 8
- 余数取上卦
- 余数为 0 时按 8

### 下卦算法

- 第二位数字 ÷ 8
- 余数取下卦
- 余数为 0 时按 8

### 先天八卦数字对应

- 1：乾
- 2：兑
- 3：离
- 4：震
- 5：巽
- 6：坎
- 7：艮
- 8：坤

### 八卦与五行对应

- 乾：金
- 兑：金
- 离：火
- 震：木
- 巽：木
- 坎：水
- 艮：土
- 坤：土

### 本卦含义

- 本卦代表事情开始时的状态

## 第三步：做计算，算变卦

- 两位数字相加
- 再 ÷ 6
- 余数取动爻
- 余数为 0 时按 6
- 自下而上数第几爻动
- 阴变阳，阳变阴，得到变卦

### 变卦含义

- 变卦代表事情的结果或后续走向

## 第四步：做解析

这里是判断的关键，不只是报卦名。

要同时看：

- 本卦
- 变卦
- 体
- 用
- 八卦五行
- 五行生克

### 当前插件里的入门解释策略

- 体：默认取本卦上卦
- 用：默认取本卦下卦
- 体用同气：偏吉
- 体生用：可成，但自己更费力
- 用生体：外部条件对自己有帮助
- 体克用：能控局，但过程未必轻松
- 用克体：偏凶，要谨慎

## 插件如何使用这份方法论

- `docs/manual.md` 负责简版规则
- 本文件负责方法论参考
- `scripts/meihua_calc.py` 负责确定性计算
- `image2 / image_gen` 只负责把过程讲得更直观、更好看
