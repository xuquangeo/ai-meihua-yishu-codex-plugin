# 八卦图示固定参考

本文件用于把八卦的图示规则彻底写死，避免后续在画图时混乱。

## 基础规则

- `阳`：一整横
- `阴`：中间断开，两短横

## 说明

下面的图示顺序统一按：

- `从上到下` 看三爻图

## 八卦固定图示

### 乾卦

- 三阳
- 上到下：阳 / 阳 / 阳

### 兑卦

- 上到下：阴 / 阳 / 阳

## 强制要求

- 不允许只写卦名，不校对图示
- 不允许只画图示，不校对卦名
- 每次都要同时对齐：
  - 卦名
  - 图示
  - 五行

## 对 image2 的要求

如果使用 `image2 / image_gen` 生成教学卡：

- 必须把这里的八卦图示当成硬约束
- 不允许模型自己脑补三爻结构
- 发现卦名和图示不一致时，以这里的图示规则为准，重新修正
