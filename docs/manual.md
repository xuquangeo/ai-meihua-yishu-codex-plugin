# 梅花易数插件规则摘要

本插件当前采用你提供的《梅花易数入门》手册中的入门规则。

更完整的方法论参考见：

- `./methodology-reference.md`

## 基本流程

1. 明确要占的事情。
2. 取两位数字。
3. 第一位数字除以 8，余数取上卦，余 0 按 8。
4. 第二位数字除以 8，余数取下卦，余 0 按 8。
5. 两数相加后除以 6，余数取动爻，余 0 按 6。
6. 自下而上数动爻，阴阳反转，得到变卦。
7. 结合体用与五行关系做吉凶判断。

## 八卦数字映射

1. 乾
2. 兑
3. 离
4. 震
5. 巽
6. 坎
7. 艮
8. 坤

## 八卦五行映射

- 乾：金
- 兑：金
- 离：火
- 震：木
- 巽：木
- 坎：水
- 艮：土
- 坤：土

## 当前解释策略

- 体用判定：`不变为体，变化为用`
- 动爻在 `1-3` 爻：下卦在变，所以上卦为体、下卦为用
- 动爻在 `4-6` 爻：上卦在变，所以下卦为体、上卦为用
- 先输出体用五行关系
- 再输出本卦到变卦的五行变化
- 最后给出入门级吉凶提示
