# 梅花易数教学图校验清单

本文件用于校验最终教学图是否能交付。

目标不是“看着差不多”，而是检查关键对应有没有画错。

## 一票否决项

以下任一项错误，整张图判定为不合格：

- 本卦卦名错
- 变卦卦名错
- 上卦图形错
- 下卦图形错
- 本卦上下组合错
- 动爻位置错
- 动爻阴阳变化方向错
- 变卦图形错
- 八卦名称与图形对照错
- 八卦与五行对应错
- 五行生克图缺元素或关系错

## 必查项

### 1. 问题与数字

- 问题文字是否正确
- 数字是否正确

### 2. 本卦

- 第一位数字对应的上卦对不对
- 第二位数字对应的下卦对不对
- 上卦图形对不对
- 下卦图形对不对
- 上卦是否明确画出了 3 条爻线，不多不少
- 下卦是否明确画出了 3 条爻线，不多不少
- 本卦是不是“上卦在上、下卦在下”
- 本卦上半部分是否与已展示的上卦图形完全一致
- 本卦下半部分是否与已展示的下卦图形完全一致
- 本卦六爻是否明确画出了 6 条爻线，不多不少

### 3. 变卦

- 是否先做了“两数相加”
- 是否再“除以 6”
- 余数是否正确
- 动爻位置是否正确
- 动爻是否按正确方向变化
- 变卦名字是否与图形一致

### 4. 八卦图形对照区

必须检查这 8 个是否同时正确：

- 乾：阳 / 阳 / 阳
- 兑：阴 / 阳 / 阳
- 离：阳 / 阴 / 阳
- 震：阴 / 阴 / 阳
- 巽：阳 / 阳 / 阴
- 坎：阴 / 阳 / 阴
- 艮：阳 / 阴 / 阴
- 坤：阴 / 阴 / 阴

并且每一个三爻卦都必须满足：

- 一卦 = 3 条爻线
- 不能偷画成 2 条粗线
- 不能把两条相邻爻合并成一条

### 5. 八卦与五行

- 乾：金
- 兑：金
- 离：火
- 震：木
- 巽：木
- 坎：水
- 艮：土
- 坤：土

### 6. 五行生克图

必须完整包含：

- 金
- 水
- 木
- 火
- 土

并且要同时有：

- 相生链
- 相克链

## 校验结论格式

校验结果只分两种：

- `通过`
- `不通过`

如果不通过，必须明确指出：

1. 错在哪
2. 是哪一类对应错了
3. 应该改成什么
